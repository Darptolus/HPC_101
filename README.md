# HPC_101
Examples for class

## Installation (Linux):
- Install programming tools (gcc, make)

    `sudo apt-get install build-essential`

- Install openmp

    `sudo apt-get install libomp-dev`

- Install MPI

    `sudo apt install mpich`

## Running the examples
- Change to the folder directory
    
    `cd 01_test`

- Compile
    
    `make`

- Run
    
    `make run`
